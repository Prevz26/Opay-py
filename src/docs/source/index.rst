Welcome to the Python client library for Opay!

**Opay** is a digital payment service and financial technology (fintech) company primarily operating in Africa. Opay offers a range of financial services, including mobile money, bank transfers, and bill payments, making it one of the most reliable fintech services.

Opay enables users to make fast and secure financial transactions. It also supports merchants by providing a payment gateway and financial management tools.

Opay provides well-secured and structured APIs, making it a great choice for businesses that need to handle online transactions and payments efficiently.

**The Opay-py library** is a Python client for interacting with the Opay API. It simplifies creating payments, managing transactions, and communicating with the Opay API.
